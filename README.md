# customer_behavoir_analyis
🔍 Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI to uncover actionable insights. It covers the complete data analytics workflow—from data loading and cleaning to visualization and reporting.

The goal is to understand customer purchasing patterns, improve decision-making, and provide business recommendations through data-driven insights.

📁 Dataset
Dataset contains customer purchase details such as demographics, product categories, spending, and frequency.
Data is processed and cleaned using Python before analysis.
🛠️ Tools & Technologies
Python – Data cleaning & EDA (Pandas, NumPy, Matplotlib, Seaborn)
SQL (MySQL/PostgreSQL/SQL Server) – Data querying and analysis
Power BI – Interactive dashboard creation
Excel – Initial data exploration
Gamma / PowerPoint – Final presentation
